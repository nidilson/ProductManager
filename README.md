# Catálogo de Productos
## Descripción
Catálogo de Productos es una aplicación web desarrollada en ASP.NET MVC (.NET Framework 4.8) que permite administrar un listado de productos tecnológicos almacenados en un archivo JSON. La aplicación ofrece funcionalidades para crear, visualizar y eliminar productos mediante una interfaz web responsive utilizando Bootstrap.

## Tecnologías utilizadas
- ASP.NET MVC 5
- .NET Framework 4.8
- C#
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- AJAX
- JSON como almacenamiento de datos
- Visual Studio 2026 Community

## Instrucciones para ejecutar el proyecto
1. Clonar o descargar el proyecto.
2. Abrir la solución (.sln) con Visual Studio 2026 Community.
3. Restaurar los paquetes NuGet si es necesario.
4. Verificar que el archivo productos.json exista dentro de la carpeta App_Data.
5. Establecer el proyecto como proyecto de inicio.
6. Ejecutar la aplicación utilizando IIS Express (F5 o Ctrl + F5).
7. Acceder al navegador para utilizar las funcionalidades de administración de productos.

## Funcionalidades
- Listado de productos.
- Agregar nuevos productos.
- Visualizar los detalles de un producto mediante un modal con AJAX.
- Eliminar productos con confirmación.
- Almacenamiento de la información en un archivo JSON.
